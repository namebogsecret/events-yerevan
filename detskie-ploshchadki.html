Новые детские площадки Еревана

Тематические (Керон) + дворовые/мини-поля по районам, 2024–2026. Кликните карточку — карта подлетит к метке.
⭐ звёзды на карте = самые большие и прикольные (топ-выбор).

На карте 344 точки площадок (OSM/Overpass, точные координаты), раскрашены по возрасту: ● ярко-зелёные = новые (за последние 4 года), ● бледные = старые. Тумблер справа вверху. 🚂 Красная линия — детская ж/д в ущелье.
«Новизна» = дата появления/правки на карте OSM (прокси, не точная дата постройки).

🏠 Дворовые и прочие новые (по районам)

Малые дворовые площадки + мини-поля + новые парки от мэрии (Шенгавит, Эребуни, Аван…). Выборка задокументированных в прессе; всего по городу за 2024 г. благоустроено ~309 дворов — точных координат по каждому нет, см. кнопки «живых карт» выше.

🟦 Арабкир / коридор Барекамутюн → центр

Прямо у Барекамутюна большой тематической нет — они все в центре (вниз по Баграмяна). Здесь: новые локальные в Арабкире + хорошие существующие парки в коридоре.

Точность: 🟢 344 OSM-точки и 🚂 ж/д — точные координаты. Кураторские метки (тематические/дворовые/Арабкир): №3 Изобретатели — точная (Wikidata); большие №1–2 и дворовые — привязка к парку/дому (±50–200 м), уточняйте кнопкой «Google Maps». Источник OSM-слоя: Overpass API (leisure=playground), bbox Еревана.
